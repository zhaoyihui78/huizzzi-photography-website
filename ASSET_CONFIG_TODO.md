## 填写说明

1. **所有底层文件名保持不变**（如 beijing01.jpg），只修改界面上显示的文字。
2. **alt**：图片的文字描述，用于无障碍访问和 SEO，建议写具体场景，如"故宫角楼雪景"。
3. **caption**：图片标题，展示在图片下方，不填则只显示 alt。
4. **不想修改的项目**：直接把整行删掉，或者保留"（请在这里填写）"我会跳过它。
5. **EXIF 格式参考**：
   - Camera: `Sony A7R IV` / `Canon EOS R5` / `Leica M6`
   - Lens: `24-70mm f/2.8 GM` / `16-35mm f/2.8`
   - Aperture: `f/8` / `f/2.8`
   - Shutter: `1/125s` / `15s`
   - ISO: `ISO 100` / `ISO 3200`
6. **胶片日期格式**：`2024.03.12` 或 `2024.03`
7. **视频季节/月份**：按实际季节填写，如季节"冬"、月份"正月"或"一月"
